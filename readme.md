| Часть JSON-объекта | Кто отвечает | Почему так? |
|---|---|---|
| `{` | SDK | Начало объекта |
| `"prompt": "` | SDK | Постоянный ключ |
| `What is the sum...` | LM (Модель) | Данные из INPUT_JSON |
| `", "name": "` | SDK | Технический переход |
| `fn_add_numbers` | LM (Модель) | Выбор функции из Registry |
| `", "parameters": {` | SDK | Технический переход |
| `"a": ` | SDK | Имя аргумента из FunctionScheme |
| `2.0` | LM (Модель) | Извлечение числа из промпта |
| `, "b": ` | SDK | Разделитель для следующего параметра |
| `3.0` | LM (Модель) | Извлечение второго числа |
| `}}` | SDK | Закрывающие скобки |
